#  Student Data Mining & Analysis System

##  Overview
This project focuses on analyzing student data to uncover patterns and insights using data mining techniques. It combines text analysis, clustering, and visualization to better understand student profiles, skills, and relationships.

---

##  Objectives
- Analyze structured and unstructured student data  
- Discover patterns and relationships between students  
- Build an interactive system for querying and exploring data  

---

##  Features
- 🔍 **Text Mining**: Uses TF-IDF to search and rank student bios based on user queries  
- 📊 **Clustering**: Groups students based on similarities using machine learning techniques  
- 🌐 **Graph Analysis**: Visualizes relationships between students, skills, and certifications  
- 🧠 **Pattern Discovery**: Extracts meaningful insights from the dataset  

---

##  Tech Stack
- Python  
- Pandas & NumPy  
- Scikit-learn  
- NLTK  
- Matplotlib & Seaborn  

---

##  How to Run
1. Open the Jupyter Notebook  
2. Run all cells step by step  
3. Enter different queries to test the system  
4. Explore clustering and visualization results  

---

##  Results
- Successfully identified patterns and grouped students into clusters  
- Implemented a working text search system for student bios  
- Visualized relationships using graph-based techniques
